# Intro

Extension for pi-mono https://github.com/badlogic/pi-mono that tracks your tps/s

# Installing

- npm (not yet :D)
- pi install git:github.com/tonussi/pi-extension-token-speed-monitor.git

# Guide

- mkdir -p ~/.pi/extensions
- cd ~/.pi/extensions
- Clone repo inside extensions
- either execute pi and enable /tps
- Or pi -e index.ts to test it
